This site is a compilation of my notable programs displayed chronological order. Each of these were designed in order to reinforce a certain aspect of coding, learned through either Computer Problem Solving, AP Computer Science, or forums.



Websites


Language: HTML/CSS/Javascript

Made Within: Visual Studio Code

The creation of websites was taught in my Computer Problem Solving class in order to reinforce programming logic. I now use my knowledge in the subject in order to create personal websites to share my work, such as this one. Should the title be clicked of any of the programs a copy will be downloaded and should the project's image be quickable it links to a page with an available demo.
